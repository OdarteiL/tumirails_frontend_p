# User Stories

## Customer Stories

### Site Management
- As a customer, I want to register my property site so I can get solar estimates
- As a customer, I want to add multiple sites so I can manage different properties
- As a customer, I want to upload site images so installers can assess my property

### Energy Estimation
- As a customer, I want to add my appliances and usage patterns so I can estimate my energy needs
- As a customer, I want to see my daily/monthly energy consumption so I can understand my requirements
- As a customer, I want to get hardware recommendations so I know what equipment I need

### Project Management
- As a customer, I want to initiate a solar project so I can get my system installed
- As a customer, I want to track project progress so I know when installation will be complete
- As a customer, I want to view project costs so I can budget accordingly (ADR later)

### Payment
- As a customer, I want to choose payment plans so I can pay in installments
- As a customer, I want to make payments online so I can pay conveniently
- As a customer, I want to track payment history so I know what I've paid

## Provider Stories

### Hardware Management
- As a provider, I want to add hardware products so customers can purchase them
- As a provider, I want to set pricing and specifications so customers have accurate information
- As a provider, I want to manage inventory so I know what's available
- As a provider, I want to upload product images so customers can see the equipment

### Business Management
- As a provider, I want to view project orders so I can fulfill hardware requests
- As a provider, I want to track payments so I know when I'll be paid
- As a provider, I want to manage my company profile so customers can find me

## Installer Stories

### Profile Management
- As an installer, I want to set my operational area and time availability so customers can find me
- As an installer, I want to upload my license and certifications so customers trust me
- As an installer, I want to manage my availability so I can control my workload

### Project Execution
- As an installer, I want to view available projects so I can bid on work (Incentive based feature Post MVP) - Customers will be able to rate the Installers
- As an installer, I want to accept project assignments so I can start work
- As an installer, I want to update project status so customers know progress
- As an installer, I want to upload installation photos so I can document my work

### Payment
- As an installer, I want to track my earnings so I know what I'll be paid
- As an installer, I want to receive payments automatically so I get paid on time

## Verifier Stories (Post MVP)

### Verification Process
- As a verifier, I want to view completed installations so I can schedule inspections
- As a verifier, I want to document verification results so quality is assured
- As a verifier, I want to upload verification photos so there's proof of inspection
- As a verifier, I want to approve/reject installations so only quality work is accepted

### Business Management
- As a verifier, I want to manage my schedule so I can plan inspections
- As a verifier, I want to track my earnings so I know what I'll be paid

## Admin Stories

### User Management
- As an admin, I want to manage user accounts so I can control platform access
- As an admin, I want to verify installer/verifier credentials so only qualified professionals work
- As an admin, I want to suspend and deactivate/reactivate problematic users so I can maintain platform quality

### System Management
- As an admin, I want to manage appliance categories so the system stays organized
- As an admin, I want to manage hardware types so providers can categorize products
- As an admin, I want to view platform analytics so I can track business performance

### Content Management
- As an admin, I want to manage default appliances so customers have options
- As an admin, I want to set system configurations so the platform operates correctly
